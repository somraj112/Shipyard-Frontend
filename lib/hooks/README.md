# Custom Hooks

> Reusable React hooks for the Shipyard frontend

Hooks here should be generic and composable. Feature-specific hooks can live in this directory with descriptive names.

## Planned Hooks

| Hook | Description |
|------|-------------|
| `useHealthStatus` | Polls the API gateway health endpoint |
| `useAuth` | Authentication state management |
| `useWorkflows` | Workflow data fetching and caching |
